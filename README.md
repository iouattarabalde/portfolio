# ismaelob.com

Portfolio d'Ismael OB, coloriste basé à Montréal. Site statique, hébergé sur GitHub Pages.

## Structure
- `index.html` — page principale (hero reel + grille de travaux + contact)
- `project.html` — gabarit à dupliquer pour chacun des 9 projets (project-01.html, project-02.html, etc.)
- `intake-form.html` — formulaire d'intake client (envoie par mailto, pas de backend)
- `style.css` — feuille de style partagée
- `assets/` — stills et médias des projets

## À faire
1. Uploader le reel sur Vimeo, remplacer `YOUR_VIMEO_ID` dans `index.html`
2. Dupliquer `project.html` en `project-01.html` ... `project-09.html`, remplir titre/client/type/année/crédits
3. Ajouter les stills dans `assets/`
4. Mettre à jour les liens `href="project.html"` dans la grille de `index.html` vers chaque page projet
